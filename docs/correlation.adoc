== Score Meaning

[cols="1,4",options="header"]
|===
| Score | Interpretation
| 0 | One signal normal → no concern
| 1 | Weak anomaly, local noise likely
| 2 | Same subsystem failure (broadcast / cellular) — probably local or maintenance
| 3 | Possible regional infrastructure issue
| 4 | Independent subsystems fail → **strong anomaly**
| 5 | Extremely unlikely coincidence → **possible systemic collapse**
|===
