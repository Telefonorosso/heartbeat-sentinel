== Score Meaning

[cols="1,4",options="header"]
|===
| Score | Interpretation
| 0 | One signal normal → no concern
| 1 | Weak anomaly, local noise likely
| 2 | Same subsystem failure (broadcast / cellular) — probably local or maintenance
| 3 | Possible regional infrastructure issue
| 4 | Independent subsystems fail → **strong anomaly**
| 5 | Extremely unlikely coincidence → **possible systemic collapse**
|===

> *Truth emerges from independent failures persisting over time.*

== Operational Logic

* No single signal → **never** panic
* Noise dies fast — **persistence matters**
* Cross-domain failure = truth signal
* Radio + space + aviation + internet down → **civilization heartbeat flatline**

== Rule of Thumb

[cols="2,3",options="header"]
|===
| Condition | Meaning

| Few isolated 2-scores | Normal noise / local glitch
| ≥3 events scoring ≥3 | Regional trouble
| ≥2 events scoring ≥4 + persistence | Serious infra anomaly
| ≥5 domains silent across radio + GNSS + ADS-B + Internet | **Critical: world heartbeat silent**
|===
